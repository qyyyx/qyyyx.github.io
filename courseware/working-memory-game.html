1什么是工作记忆

工作记忆是大脑临时存储和操作信息的能力，就像电脑的内存一样。它允许我们在短时间内保持信息活跃，以便进行复杂的认知任务，如理解文本、解决问题或做出决策。工作记忆是我们日常学习、工作和生活的基础认知能力之一。

2什么是工作记忆广度

工作记忆广度（Working Memory Span）是指大脑一次能够同时处理和存储的信息单元数量。1956年，心理学家乔治·米勒在论文《神奇的数字7±2》中提出，人类工作记忆的平均容量约为7个信息单元，这就是著名的"魔法数字"理论。

简单来说：工作记忆广度就是你在脑海中同时"记住"和"操作"信息的最大数量。比如记住一串电话号码、在心算时保持中间结果等，都依赖工作记忆。

3工作记忆广度的意义

工作记忆广度与许多重要能力密切相关：

学习能力：工作记忆广度越大，学习新知识的速度越快，理解能力越强。

阅读理解：阅读时需要同时记住已读内容、理解当前句子、预测后续内容。

问题解决：复杂问题需要多步骤推理，每一步都需要在工作记忆中保持中间结果。

决策能力：做决策时需要权衡多个选项和因素。

4黑猩猩记忆实验

2007年，日本京都大学灵长类研究所的松泽哲郎教授进行了一项著名的研究，研究发现，黑猩猩在记忆测试中的表现竟然优于人类！特别是名叫"步"的年轻黑猩猩，能够以90%的准确率记住9个数字的位置。

科学家认为，这可能与黑猩猩的"遗觉象"（eidetic memory）有关——它们能够保持异常清晰的视觉表象记忆。

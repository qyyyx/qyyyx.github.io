点动成线

点动成线：从一段弧到复合轨迹

一个动点经过的路径就是它的「轨迹」。先看单点绕中心，再看会翻滚、会绕圈的图形，点画出的轨迹。

先预测

把角度从 0 拉到 360°，点 A 的轨迹会是——

选一个猜想，再动手验证。
6
120°
实时 · 弧长公式
弧长 = (角÷360) × 2πr
先拖角度观察，准备好后点「显示结论」。
点动成线。一个点绕定点旋转，轨迹是圆（或圆弧）；转满 360° 就是整圆。
弧长 = (n÷360) × 2πr　　整圆周长 = 2πr
半径越大、转角越大，轨迹越长——长度只由 r 和转角 n 决定。
线动成面

线动成面：线 / 圆扫过的面积

一条线、一个圆动起来会扫出一片面积。看三角形的边、绕方滚动的圆、栓羊的绳分别扫出多少。

实时 · 扫过面积（π取3.14）
—
先播放看面积怎么扫出来，再点「显示结论」。
线动成面。
三角形绕 C 转 90°：AB 边扫过 = 圆环扇形 = ¼π(R²−r²)
圆绕正方形：扫过 = 边带 + 拐角（外 8sr+4πr²｜内 s²−(s−2r)²−(4r²−πr²)）
栓羊：主扇形 + 绳绕过每个角后剩余绳长的小扇形相加
关键都是：找到动点能到达的最远/最近边界，面积 = 一串扇形/圆环之和。
面动成体

平面图形绕轴转一周，长成立体

把一个平面图形绕直线 l 转 360°，它会扫出一个立体。先连一连，再亲手转、算体积。

0°
实时 · 体积（π取3.14）
—
转动每个图形看它长成什么，再点「显示结论」。
面动成体。矩形→圆柱、直角三角形→圆锥、直角梯形→圆台、半圆→球。
圆柱 V=πr²h　圆锥 V=⅓πr²h　复合体 = 圆柱 + 圆锥
预习5：矩形(r5·h10)转出圆柱 + 上方三角转出圆锥 = 3.14×5²×10 + ⅓×3.14×5²×6 = 785+157 = 942 cm³。
硬币旋转悖论 ★

绕一圈，硬币自转几圈？

一枚硬币贴着另一形状外侧滚一圈回到原位。它自己转了几圈？多数人会猜错。

先预测

两枚一样大的硬币，一枚绕另一枚外侧滚一圈，动的那枚自转了——

选一个猜想，再亲手滚一圈数数。
实时 · 自转计数
自转圈数 = 公转贡献 + 滚动贡献
先预测再滚一圈，盯住硬币上的红点数自转，再点「显示结论」。
多出来的「+1」来自公转。沿路滚动让硬币转「路程 ÷ 自身周长」圈，而绕着中心走一圈本身又额外贡献 1 圈。
外侧绕一周：n + 1 圈　　内侧：n − 1 圈　（n = 两者周长之比）
等大两圆 n=1 → 外滚自转 2 圈；绕多边形：圆心路程 = 周长 + 2πr，自转 = 周长÷自身周长 + 1。
滑轮问题 ⚙️

定滑轮 vs 动滑轮：绳子要拉多长？

轮子转，绳子动。定滑轮只改方向，动滑轮能省力——代价是绳子要拉更长。亲手提一提。

先预测

用动滑轮把重物提高 1 米，你要拉动多长的绳子？

选一个猜想，再亲手拉一拉。
实时 · 绳长与省力
—
先拉一拉，比较「拉的绳长」和「物体升高」，再点「显示结论」。
承重几段绳，就省几倍力、拉几倍绳。
拉动绳长 = 承重段数 n × 提升高度 h　　拉力 = 重力 ÷ n
定滑轮 n=1（只改方向，不省力，拉 1 倍绳）；动滑轮 n=2（省一半力，拉 2 倍绳）；滑轮组 n 段（省 n 倍力，拉 n 倍绳）。功不会凭空省：省了力就多拉绳。
